## passwordcrack here must be in zip file format



### how to installation:

###### install git and python
```yml
    sudo apt install git python
```
###### install zipfile
```yml
    sudo pip install zipfile
```
###### clone script from this repository
```yml
    git clone https://github.com/KuMarayana/passwordcrack.git
```
###### point to directory passwordcrack
```yml
    cd passwordcrack/
```
###### run script(start program)
```yml
    sudo ./passwordcrack.py
```
###### _Or_
```yml
    sudo python passowordcrack.py
```
